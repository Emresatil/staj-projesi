# Staj Projesi - Yetkinlikler ve Implementation Plan Taslağı

## 1. Proje Hedefleri
* Staj süresi boyunca atanacak görevleri, mevcut teknik bilgi birikimimi kullanarak Agile prensiplerine uygun olarak tamamlamak.
* İhtiyaç duyulması halinde hem oyun motoru dinamiklerini hem de mobil uygulama arayüzlerini projeye entegre edebilecek esnek bir yapıda çalışmak.

## 2. Neler Yapabilirim? (Teknik Altyapı)
Şirket tarafından net bir proje atanana kadar, aşağıdaki alanlarda geliştirme yapabileceğim bir altyapım bulunmaktadır:

* **Mobil Uygulama:** React Native (Expo Go, Redux, Reanimated v2) kullanarak hızlı ve akıcı mobil arayüzler inşa edebilir; veri listeleme, filtreleme veya arama fonksiyonları (wiki tarzı uygulamalar vb.) geliştirebilirim.
* **Backend & Veritabanı:** Gerekli durumlarda projenin arka planı için NestJS ve SQL kullanarak temel veritabanı bağlantılarını ve mimarisini kurabilirim.
* **Performans Takibi:** Geliştirilen ürünlerde Performance Monitor ve React DevTools Profiler kullanarak FPS ve bellek optimizasyonlarını analiz edebilirim.

## 3. Yol Haritası
1. **Aşama:** GitHub repomun oluşturulması ve Git workflow (Conventional Commits) standartlarının ayarlanması. *(Tamamlandı)*
2. **Aşama:** Yöneticilerim tarafından yukarıdaki yetkinliklerime (veya şirketin güncel ihtiyaçlarına) uygun proje/modül atamasının yapılması.
3. **Aşama:** Kodlama sürecinin başlaması, staj defterine işlenecek olan günlük commit'lerin atılması ve Daily Standup toplantılarında raporlama yapılması.
